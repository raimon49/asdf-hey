# asdf-hey

## Installation

```bash
$ asdf plugin add hey https://github.com/raimon49/asdf-hey.git
```

## Usage

```bash
# Install any version of hey
$ asdf install hey v0.1.4

# Set the version of hey you want
$ asdf global hey v0.1.4
$ hey --version
hey version v0.1.4
```
